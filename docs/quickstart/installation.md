# Installation

## Prerequisites 

* [Docker](https://docs.docker.com/get-docker/) is required to use bisi.

* Awscli and AWS credentials are required to use the AWS Batch features.

    ```bash
    pip install awscli
    aws configure
    ```

## Install bisi

Install bisi from [pypi](https://pypi.org/project/bisi/)

```bash
pip install bisi
```
